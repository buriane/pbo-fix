# Workshop Registration Web App with Play Framework 1.6

This is a simple web application built with Play Framework 1.6 for workshop registration. The application allows you to perform CRUD operations on participants and workshops.

## Getting Started

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/your-username/workshop-registration-app.git
   cd workshop-registration-app

2. **Install Play Framework 1.6:**

   Make sure you have Play Framework 1.6 installed on your machine. You can download it from the [official website](https://www.playframework.com/download).

3. **Run the Application:**

   ```bash
   play run
   ```

   Open your browser and go to [http://localhost:9000](http://localhost:9000) to access the application.

## Features

### Participants

- **View Participants:**
  - Navigate to the Participants section to view a list of registered participants.
- **Add Participant:**
  - Click on the "Add Participant" button to register a new participant.
- **Edit Participant:**
  - Click on the "Edit" button next to a participant's details to update their information.
- **Delete Participant:**
  - Click on the "Delete" button next to a participant's details to remove them from the list.

### Workshops

- **View Workshops:**
  - Navigate to the Workshops section to view a list of available workshops.
- **Add Workshop:**
  - Click on the "Add Workshop" button to create a new workshop.
- **Edit Workshop:**
  - Click on the "Edit" button next to a workshop to modify its details.
- **Delete Workshop:**
  - Click on the "Delete" button next to a workshop to remove it.
